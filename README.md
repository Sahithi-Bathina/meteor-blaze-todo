# 🚀 Meteor Blaze Todo Application

A modern and interactive Todo Management Application built using **Meteor.js** and **Blaze Templates**.

This project was developed as part of a technical assessment to demonstrate:
- Meteor.js fundamentals
- Blaze template rendering
- MongoDB integration
- Reactive UI handling
- Drag-and-drop functionality
- Modern frontend UI/UX practices

---

## ✨ Features

- ✅ Add Tasks
- ✅ Delete Tasks
- ✅ Mark Tasks as Complete
- ✅ Task Categories
- ✅ Drag-and-Drop Reordering
- ✅ Persistent Task Order
- ✅ Responsive Modern UI
- ✅ Confetti Celebration on Task Completion
- ✅ Empty State Handling
- ✅ Interactive Hover Effects

## 🎯 Key Functionalities

### 📂 Task Categories
Tasks can be categorized into:
- Work
- Placement
- Personal
- Urgent
- General

### 🔀 Drag-and-Drop Reordering
Tasks can be reordered interactively using drag-and-drop functionality powered by **SortableJS**.

### 🎉 Completion Celebration
A confetti animation is triggered whenever a task is marked as completed.

### 💾 Persistent Data
Tasks and their order are stored using MongoDB and persist after page refresh.

## 📦 Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/Sahithi-Bathina/meteor-blaze-todo.git
```

### Navigate to Project Directory

```bash
cd meteor-blaze-todo
```

### Install Dependencies

```bash
meteor npm install
```

### Run the Application

```bash
meteor run
```

### Open in Browser

```text
http://localhost:3000
```
